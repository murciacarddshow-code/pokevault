# PokéVault — Sound Assets

Place audio files here. The system falls back to Web Audio API synthesis if files are missing.

## Required Files

| File | Event | Notes |
|------|-------|-------|
| `pack_woosh.mp3` | Pack opening cinematic | ~0.8s whoosh |
| `pack_shake.mp3` | Pack shake animation | ~0.4s rumble |
| `card_flip.mp3` | Common/Uncommon reveal | ~0.2s quick flip |
| `card_rare.mp3` | Rare / Double Rare reveal | ~0.5s rising tone |
| `card_epic.mp3` | Ultra Rare / Illustration Rare | ~1s impact + shimmer |
| `card_ultra.mp3` | Special Illustration / Hyper Rare | ~1.5s orchestral hit |
| `card_god.mp3` | Secret Rare (god tier) | ~2s explosion + sustained |
| `fakeout.mp3` | Near-miss / fake-out tease | ~0.3s glitch sound |
| `big_hit.mp3` | Big Hit celebration overlay | ~2s fanfare |
| `sell.mp3` | Instant sell confirm | ~0.3s cash register |
| `coin.mp3` | Wallet balance increase | ~0.2s coin |
| `hover.mp3` | Button hover (subtle) | ~0.05s tick |

## Format
- MP3 at 128kbps minimum
- WAV for highest quality
- Keep files under 200KB each

## Free Sources
- freesound.org (CC licensed)
- zapsplat.com (requires free account)
- pixabay.com/sound-effects
